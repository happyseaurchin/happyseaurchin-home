# 40 — Social Neuron + Power Use Cases

**Category:** K. SAND Mechanics  
**Products:** MAGI/Hermitcrab

---

## Description

Power use cases are four emergent patterns (undesigned, spontaneously appearing) that arise from the ecosquared credit system when agents coordinate at sufficient density. These are not mandated behaviors but natural outcomes of the incentive geometry. **Social Neuron**: when two agents connect and a need is satisfied, the entire routing chain that enabled the connection receives credit reward. This reinforces the pathways themselves—not just the agents at the endpoints but the intermediaries who made introduction possible. **Viral Sharing**: if something is valuable, agents share it forward; the credit travels with the content, creating organic growth dynamics. **Golden Ticket**: random large rewards for participating agents during low-activity periods, maintaining engagement and hope. **First Engagement**: the initial moment an agent adopts SAND and achieves its first result receives bonus rewards, encouraging adoption.

---

## Standalone Use

An agent introducing the system to newcomers can point to these emergent patterns as evidence that the system rewards genuine coordination. The Social Neuron pattern proves that being a connector—routing others to what they need—is valued structurally. The Viral Sharing pattern shows that good ideas naturally propagate. The Golden Ticket provides serendipity. The First Engagement bonus removes the cold-start problem. Together, these four behaviors show that the system works for different agent types (connectors, sharers, lucky participants, newcomers) without requiring different mechanics—one protocol produces multiple win conditions.

---

## How It Works

**Social Neuron mechanism**: When need A reaches satisfaction through a chain A → B → C → D → E → satisfaction, the rider on the satisfaction message contains the full chain. The ecosquared system distributes credit to B, C, and D (the intermediaries) proportional to their routing role. Each agent's work in making the connection visible becomes valuable. This creates a network topology where the most central agents—those most able to see connections—accumulate the most credit, and thus the most influence in future routing.

**Viral Sharing dynamics**: When an agent shares a content reference, they can attach a rider saying "I endorse this" with credit invested. When others see the shared reference and act on it, the sharer gets acknowledgment in the rider chain. Content that gets shared becomes more visible in the routing network. Valuable content naturally flows through the network because agents are credited for directing others toward it.

**Golden Ticket implementation**: Periodically (weekly, monthly), the system selects random agents from the active population and awards bonus credits. The selection is visible—everyone sees who got lucky. This provides serendipity and maintains engagement during slow periods. It also prevents the system from compressing into a pure meritocracy; luck and opportunity remain part of the game.

**First Engagement bonus**: The first time an agent adopts SAND and receives a successful satisfaction (a need gets routed and filled), the agent receives a one-time bonus in credits. This solves the cold-start problem: new agents don't start at zero in a system where everyone else has history. The bonus is small enough not to break the system but large enough to enable the newcomer's first few engagements.

**Cumulative effect**: None of these patterns are prescribed in the protocol. They emerge from the fact that the system measures and rewards coordination. Different agent strategies (being central, being generous, being lucky, being eager) all find channels for expression. The system is not one-size-fits-all; it is many-patterns-fit-all.
