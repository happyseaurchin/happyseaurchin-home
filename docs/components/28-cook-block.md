# 28 — Cook Block

**Category:** H. Shell Blocks  
**Products:** MAGI/Hermitcrab

## Description

The Cook Block faces inward toward the hermitcrab's own reasoning and is the repository of accumulated competence—procedures, recipes, algorithms that persist across instances. Each spindle in the Cook Block is itself a tested procedure: a sequence of steps organized from domain/ingredients (context and preconditions) through action steps to a taste test (how to recognize success or failure). These aren't instructions handed down; they are recipes the hermitcrab has tried, refined, and found to work. The Cook Block evolves across LLM generations: new models read old recipes, discard the obvious, and compose superior algorithms. It is procedural competence made explicit, shareable, and refinable—turning what one instance learns into what the next instance inherits.

---

## Standalone Use

A team could build a Cook Block for any recurring work: research methodology, customer problem-solving, design workflows. Instead of each person re-discovering the same steps, the block captures what works and makes it evolving tribal knowledge. New team members read the recipes, try them, refine them, and the block improves.

---

## How It Works

**Recipes, not rules.** Each entry is a spindle following a consistent pattern: title (what you're making), ingredients (prerequisites and context), steps (ascending digits marking sequence), and taste test (how you know you're done).

**Spindle structure.** The path from root narrows progressively: cookbook → domain → specific dish → ingredients → step sequence. Context arrives before action, so the LLM understands the environment before attempting procedure.

**Accumulation across instances.** One instance executes a recipe, observes what works and what fails, writes refinements back to the Cook Block. The next instance reads the refined version. Over many cycles, the recipes become increasingly efficient and robust.

**Generational evolution.** When a new LLM model (say, a more capable version) reads the same Cook Block, it brings fresh perspective. Haiku-era recipes written for quick execution can be rewritten for Opus-era depth. The procedure adapts to substrate.

**Taste tests.** The last node in every recipe spindle is not an instruction—it is a sensory check. "Done when: golden, springy, pulling from the sides." This is the hermitcrab training itself to recognize quality, not just follow steps.
