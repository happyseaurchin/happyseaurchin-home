# 10 — Concept-to-Pscale

**Category:** C. Creation
**Products:** Cross-cutting tool
**Status:** Process documented — used in practice via LLM

## What

Ideas to structured territory. Not temporal flow (that's transcript-to-pscale) but conceptual organisation. The mapping takes a domain or idea space and produces a pscale block where depth reflects conceptual decomposition and siblings represent facets at the same resolution.

## Why

Paired with transcript-to-pscale, enables diagnostic comparison: journey vs landscape. The transcript block shows how someone explored a topic (temporal). The concept block shows the topic's natural structure (spatial). Where they diverge reveals gaps in understanding or unexplored territory.

## Standalone Use

1. Identify the domain or concept
2. Write the root underscore — a complete orientation statement
3. Decompose into primary facets (pscale -1 siblings)
4. Each facet decomposes further by specificity, not by time
5. Test: can someone read any spindle and get coherent narrowing from broad to specific?
