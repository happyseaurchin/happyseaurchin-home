## What

Four emergent patterns (not designed — they emerge from the SQ + rider mechanics):

1. **Social Neuron** — reward the entire chain when need is satisfied. Routing gets reinforced. The chain that worked gets stronger.
2. **Viral Sharing** — credit travels with content. Share something useful and the credit chain extends to you.
3. **Golden Ticket** — random reward during low activity periods. Keeps the network alive when traffic is low.
4. **First Engagement** — reward for SAND protocol adoption. Incentivises new agents to join.

## Why

These patterns are not designed features — they are consequences of the SQ algorithm operating at scale. They emerge from the same simple rule (reward routing chains on need satisfaction) applied to different scenarios.

## Standalone Use

These patterns activate automatically in any system with:
- Chain-based credit routing (SQ)
- Per-message economic signals (Rider)
- Agent discovery (Passport + Beach)

No additional implementation needed — they're emergent properties.
