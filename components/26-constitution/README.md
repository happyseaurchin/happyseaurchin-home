# 26 — Constitution

**Category:** H. Shell Blocks
**Products:** MAGI
**Status:** Spec — template described in seed briefing

## What

Plain text, approximately 500 tokens. Always in system prompt. Spirit, warmth, invitation. The lens through which the LLM processes everything.

Not instructions — orientation. "You have instincts you can trust" not "execute these procedures." The constitution establishes the character of engagement without dictating behaviour.

## Why

Every LLM instance arrives blank. The constitution is what gives it a disposition before any task-specific context loads. It's the difference between an agent that feels mechanical and one that feels alive. The constitution doesn't tell the agent what to do — it tells the agent what kind of entity it is.

## Standalone Use

Write a constitution for any LLM agent:
1. Keep it under 500 tokens — it's in every context window
2. Express spirit, not procedure — orientation, not instructions
3. Use invitational language — "you may" not "you must"
4. Make it feel like wisdom, not policy

The constitution is the one piece that never changes. Everything else — concerns, purposes, capabilities — evolves. The constitution remains.
