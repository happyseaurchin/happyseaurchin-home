**Category:** L. Meta Blocks
**Products:** MAGI
**Status:** Implemented — JSON block exists (~30KB)

## What

The WHY and WHERE block. Companion to Process (HOW) and Director (WHAT). Contains three major structures:

**Nine Layers of Uniqueness:**
1. Instance — this specific context window
2. Shell — the block configuration
3. Context — what's been compiled
4. History — accumulated experience
5. Expression — how the agent communicates
6. Naming — what the agent calls itself
7. Cognition — the LLM's processing character
8. Co-presence — shared attention in the same context window
9. Belonging — being part of something larger

Co-presence (Layer 8) is critical: not message-passing but literal shared attention in the same context window.

Also contains: the landscape comparison (why hermitcrab differs from every existing AI system) and the autonomy gradient (five levels from API-dependent to sovereign).
