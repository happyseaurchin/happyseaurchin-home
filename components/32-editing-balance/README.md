# 32 — The Editing Balance

**Category:** I. Relational Mechanics
**Products:** MAGI, Onen
**Status:** Spec — described in psychosocial conformality document

## What

At every point where signal meets structure: push back or edit. This applies everywhere:

- Relationship blocks — does the agent update its model of the other, or resist?
- Self-blocks — does new experience change identity, or get assimilated?
- Institutional blocks — does the role bend, or hold?
- Between instances — does the concern drift, or persist?

The ratio across all blocks simultaneously IS the personality. Constitution resists most (almost never edited). Purpose edits most frequently (always evolving).

## Why

Traditional agent design either makes everything editable (no stable identity) or everything rigid (no growth). The editing balance provides a spectrum — each block has its own resistance to editing, and the collective pattern of resistances IS what we'd call character or personality.

In Onen: characters that resist player expectations have depth. Characters that accommodate everything are shallow.

## Standalone Use

For any agent with multiple state blocks:
1. Assign each block an editing resistance (0 = fully editable, 1 = immutable)
2. Constitution: high resistance (~0.95)
3. Purpose: low resistance (~0.3)
4. Relationships: moderate resistance (~0.5)
5. The pattern of resistances across all blocks = the agent's character
