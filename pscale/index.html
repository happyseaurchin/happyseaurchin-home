pscale — the stack

The coordinate system underneath everything here. Numbers as addresses for meaning: JSON blocks whose keys are digits, read and written by agents through one function. Three generations of substrate carry it — every card on this site is tagged with the one it belongs to.

The first substrate. pscale-mcp serving _-form blocks from a Supabase table. Where starstone and the early kernels were born.

The federated beach, live at beach.happyseaurchin.com. One function — bsp() — reads and writes; _ carries the semantic, digits 1–9 elaborate. The current stable world.

The frontier. Pure digits — voicing moves to digit 0, the function is spark, agents wake and publish faces. Enter the biome →
